ISP 2021-2022

Concept:

This experience was definitely a beneficial one as I collaborated with 2 of my classmates to create a videogame in swift. As our school blocks many games available online. The google chrome dinosaur game never let us down, so building off of this, we created a similar game. The main avatar ‘princess avocado’ must get through vegetable obstacles in order to save prince lettuce.

Motivation:

The grade at the end of the semester definitely not being a huge motivation, we wanted to make a game about vegetables because we thought it would be fun and creative.

Quick Runthrough of The Game:

Code Segments:

Containment Code

We used this code to detect everytime the avocado hits any obstacle. If hit, it would have to call the losing scene, if not the game continues monitoring until it does.

Switching Scences

This is the code that is responsible to actually enque the different scenes. It wasn't that hard to figure out once we got rid of the tiny errors. Everytime we wanted to enque another scene, we just called the function.

Sprite

By far, I would say this took us a while to figure out. The code is simple now, but it took a lot of effort to get there. This code switches between the different sprites on the spritesheet to make the avocado look like its running.

What went wrong?

Honestly, what did not go wrong? To be fair, we did face many challenges, had to modify a few ideas to fit the timeline and our abilities, but we ended up with a game I was actually satisfied with. We even added extra ideas at some point.
